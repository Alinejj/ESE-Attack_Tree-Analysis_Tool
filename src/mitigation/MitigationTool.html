AT-AT Mitigation Analysis Tool

Apply countermeasures to attack trees and compute delta likelihood

How to Use This Tool

1

Prepare your mitigation CSV file following the format described below. Make sure node names match exactly as they appear in your attack tree.

2

Export your attack tree as a JSON file from AT-AT using File -> Export CSV, or paste the tree JSON directly. The tree must include terminal node metrics (a, t, d values).

3

Upload both files using the upload buttons below, then click Run Analysis to compute the mitigation impact.

4

Review the results: the summary shows overall delta likelihood for the whole tree, and the table shows per-node baseline and mitigated probabilities.

CSV Format Instructions

Your CSV file must follow this two-section format:

Section 1 — Metadata (Lines 1 and 2):

# Line 1: column headers (can be named anything)

scale,w_ac,w_td,w_dd

# Line 2: actual values

INT,1,1,1


Section 2 — Node Mitigations (Line 3 onward):

# Line 3: column headers (can be named anything)

node_name,AC_new,TD_new,DD_new

# Line 4 onward: actual mitigation data

Form Collaboration of PWs,5,3,2

Gain Exclusive Access to Ballots,5,4,2

Section Column Position Description Example Header Name
Metadata scale Metric scale: either INT (1-5) or FLOAT (0-1) INT
w_ac Weight for Attack Cost. Any positive number, auto-normalized 1
w_td Weight for Technical Difficulty. Any positive number, auto-normalized 1
w_dd Weight for Discovery Difficulty. Any positive number, auto-normalized 1
Node Mitigations node_name Exact terminal node name as it appears in the attack tree Form Collaboration of PWs
AC_new New Attack Cost value after mitigation is applied 5
TD_new New Technical Difficulty value after mitigation is applied 3
DD_new New Discovery Difficulty value after mitigation is applied 2
⚠️ Important rules: Column headers can be named anything but data must be in the correct order, Column 1: terminal node name, Column 2: new AC value, Column 3: new TD value, Column 4: new DD value. Node names must match exactly as they appear in the attack tree. All nodes must use the same scale. Weights must be positive numbers greater than 0. AC and TD should only increase after mitigation. DD should only decrease after mitigation.
